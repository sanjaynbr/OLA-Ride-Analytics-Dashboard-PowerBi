# Ola Ride Analytics Dashboard (Power BI)

## 📊 Overview
This project analyzes ride booking data to uncover trends in customer behavior, revenue, and service performance.

## 🔑 Key Insights
- Majority of rides are short-distance (<10 km)
- Cash & UPI dominate payment methods
- Small group of customers contributes major revenue
- Customer ratings are consistently high (~4+)
- Strong correlation between customer and driver ratings

## 📈 Features
- KPI cards (Bookings, Revenue, Customers)
- Ride distance distribution
- Customer segmentation
- Payment method analysis
- Ratings correlation scatter plot

## 🛠 Tools Used
- Power BI
- Data Modeling
- DAX
